## 1.4 风险矩阵

| 风险项 | 风险等级 | 说明 |
|--------|----------|------|
| 三权分立机制绕过 | **严重** | 核心安全功能，可能导致权限失效 |
| VPN隧道命令注入 | **严重** | 可导致设备被控 |
| 固件升级无签名校验 | **高** | 可植入恶意固件 |
| 认证绕过/会话固定 | **高** | 非法访问管理界面 |
| 敏感信息明文传输 | **中** | 密码/密钥泄露 |
| 国密算法实现漏洞 | **高** | SM2/SM4实现问题 |
| 日志篡改/删除 | **中** | 审计追踪失效 |
| 默认口令未修改 | **高** | 弱口令利用 |
