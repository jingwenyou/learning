# 英语渗透式学习计划

> 基础：CET-4 | 职业：测试工程师 | 核心理念：不加额外任务，把日常切换成英语

---

## 第一阶段：建立习惯（第 1-4 周）

### 第 1 周：只做 3 件事

| 天 | 任务 |
|----|------|
| Day 1 | 手机 + 电脑 + IDE + 浏览器全部切英文界面 |
| Day 2 | 装好「每日英语听力」App，通勤听 15 分钟 VOA 慢速英语 |
| Day 3 | 装好 Anki，下载一个 IT/软件测试英语牌组，刷 10 张卡片 |
| Day 4-7 | 重复：通勤听 + Anki 刷卡片，形成习惯 |

**本周目标：不求学会什么，只求每天都碰了英语。**

### 第 2 周：加入工作英文

在第 1 周基础上，增加：

- 写 bug 报告 / 测试用例时用英文
- 模板：

```
**Title:** Login page shows blank after wrong password
**Steps to reproduce:**
1. Open login page
2. Enter valid username
3. Enter wrong password
4. Click "Login" button
**Expected:** Show error message "Invalid password"
**Actual:** Page goes blank
**Severity:** Major
```

- Google 搜索问题时用英文关键词，优先看 Stack Overflow
- 工作中遇到的生词，截图/拍照丢微信收藏夹，周末整理进 Anki

### 第 3-4 周：加入输出练习

在前两周基础上，增加晚上 15-20 分钟：

| 日期 | 固定安排 | 内容 |
|------|---------|------|
| 周一、三、五 | 输出日 | 用英文跟 AI 聊今天工作中的事，AI 帮纠错 |
| 周二、四 | 输入日 | 读一篇英文短文（Medium / Dev.to），摘 3 个有用的句子 |
| 周末 | 放松日 | 看一集英美剧（中英双字幕），随意，不强求 |

---

## 第二阶段：稳定提升（第 2-6 个月）

### 每日节奏（固定）

#### 通勤（30-40 分钟）

| 去程 15-20min | 回程 15-20min |
|---------------|---------------|
| 听力 + 跟读 | Anki 复习 |

**听力材料分级：**

| 阶段 | 材料 | 什么时候升级 |
|------|------|-------------|
| 第 1-2 个月 | VOA 慢速英语 / ESL Pod | 能听懂 80% 不看文字 |
| 第 3-4 个月 | BBC 6 Minute English | 能听懂 70% 不看文字 |
| 第 5-6 个月 | YouTube 技术频道（Fireship 等） | - |

**跟读法（Shadowing）：**
- 听一句，暂停，模仿着说一句
- 每天 5 分钟就够
- 用通勤听力素材，不用额外找

#### 工作中（零额外时间）

- 英文写 bug 报告 / 测试用例 / 工作笔记
- 英文搜索技术问题
- 遇到生词截图收藏，周末整理进 Anki
- 有道词典查词时**听一下发音**，纠正错误读音

#### 晚上（15-20 分钟，固定节奏不动摇）

- **周一三五 = 输出**：跟 AI 英文对话，或写 3-5 句英文工作笔记
- **周二四 = 输入**：读一篇英文技术短文
- **周末 = 放松**：美剧 / YouTube，随意

### Anki 使用规则

- 每天新增不超过 **10 个词**
- 只背工作和生活中遇到的词，不背词汇书
- 卡片格式：

```
正面：regression
背面：回归测试 / We need to run regression tests before release.
```

```
正面：intermittent
背面：间歇性的 / This is an intermittent failure, hard to reproduce.
```

- 第 1 个月用现成牌组起步，之后逐步过渡到自己建卡片

## 第三阶段：突破口语（第 7-12 个月）

- 晚上输出日从文字聊天 → 语音练习（手机录音说英文，回听自己）
- 尝试用英文做技术分享（哪怕只在组内 5 分钟）
- 如果公司有英文文档需求、外企合作机会，主动接

---

## 阶段验证目标

| 时间 | 目标 | 怎么验证 |
|------|------|---------|
| 1 个月 | 习惯英文环境 | 英文界面不想切回中文 |
| 3 个月 | 工作英文基本无障碍 | bug 报告直接写英文，不用先想中文再翻译 |
| 6 个月 | 听力明显提升 | YouTube 技术视频不开字幕能懂 70% |
| 12 个月 | 能用英文工作交流 | 能用英文描述测试流程、讨论技术方案、参加英文面试 |

## 避坑指南

| 坑 | 为什么是坑 | 怎么避 |
|----|-----------|--------|
| 一开始就背单词书 | 枯燥脱离场景，两周必弃 | 只背遇到的词 |
| 追求听懂每个词 | 产生挫败感 | 听懂大意就行，猜比查重要 |
| 语法不好就不敢写 | 完美主义害死人 | `I test the button, it not work` 比不写强一万倍 |
| 攒着周末突击 | 记忆曲线不允许 | 每天 15 分钟 > 周末 3 小时 |
| 计划太满第一天就全上 | 坚持不了一周 | 逐周叠加，别一步到位 |

## 推荐工具清单

| 用途 | 工具 | 备注 |
|------|------|------|
| 背单词 | Anki | 免费（电脑/安卓），iOS 收费可用 AnkiDroid 替代 |
| 听力 | 每日英语听力 App | 有分级，VOA 慢速英语专区 |
| 查词 | 有道词典 | 查词时听发音 |
| 输出练习 | Claude / ChatGPT | 免费、不尴尬、随时可练 |
| 美剧 | 《The IT Crowd》《Modern Family》 | IT 狂人跟职业相关，摩登家庭练日常口语 |
| 技术阅读 | Medium / Dev.to / Stack Overflow | 工作中顺手读 |

---

*最重要的一句话：**每天接触 > 系统学习**。别追求完美计划，追求每天都碰了英语。*
