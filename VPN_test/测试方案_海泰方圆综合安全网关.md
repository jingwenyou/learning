### 3.5 Phase 6 - SSH底层安全检查

#### 3.5.1 系统信息

| 检查项 | 命令 | 关注点 |
|--------|------|--------|
| 系统版本 | `cat /etc/os-release` | 已知漏洞版本 |
| 内核版本 | `uname -a` | 内核漏洞 |
| 运行进程 | `ps aux` | 异常进程 |

#### 3.5.2 敏感配置

| 检查项 | 命令 | 关注点 |
|--------|------|--------|
| 用户列表 | `cat /etc/passwd` | 非授权用户 |
| 密码文件 | `cat /etc/shadow` | 弱密码 |
| SUID文件 | `find / -perm -4000` | 可提权程序 |
| 世界可写 | `find / -writable` | 权限配置 |
| SSH配置 | `cat /etc/ssh/sshd_config` | 允许密码登录 |
| crontab | `crontab -l` | 计划任务后门 |

#### 3.5.3 Web服务配置

| 检查项 | 命令 | 关注点 |
|--------|------|--------|
| nginx配置 | `find / -name nginx.conf` | 配置错误 |
| 应用配置 | `find / -name "*.conf"` | 敏感信息 |
| 密钥文件 | `find / -name "*.key" -o -name "*.pem"` | 私钥泄露 |

#### 3.5.4 VPN配置

| 检查项 | 命令 | 关注点 |
|--------|------|--------|
| VPN配置 | `find / -path "*vpn*" -name "*.conf"` | 隧道配置 |
| 证书文件 | `find / -name "*.crt" -o -name "*.cer"` | 证书内容 |
| 共享密钥 | `grep -r "sharedsecret\|psk" /etc/` | 预共享密钥 |
