### 第三步：自动化测试脚本生成

#### 3.1 验证码识别模块（AI 视觉）

基于已有的 `analyze_captcha.py` 扩展：

```python
# 核心流程：截图验证码区域 → Claude Vision API 识别 → 填入表单 → 登录
# 文件: captcha_solver.py
import anthropic, base64
from playwright.sync_api import sync_playwright

def solve_captcha(page):
    """截取验证码图片，调用 Claude Vision 识别"""
    captcha_elem = page.locator('img[class*="captcha"], .captcha-img').first
    captcha_elem.screenshot(path='/tmp/captcha.png')

    with open('/tmp/captcha.png', 'rb') as f:
        img_b64 = base64.b64encode(f.read()).decode()

    client = anthropic.Anthropic()
    resp = client.messages.create(
        model='claude-opus-4-6',
        max_tokens=50,
        messages=[{
            'role': 'user',
            'content': [
                {'type': 'image', 'source': {'type': 'base64',
                 'media_type': 'image/png', 'data': img_b64}},
                {'type': 'text', 'text': '请识别图中的验证码字符，只输出字符，不要其他内容'}
            ]
        }]
    )
    return resp.content[0].text.strip()
```

#### 3.2 三角色登录基础框架

```python
# 文件: base_login.py
ACCOUNTS = {
    'system_admin': {'user': '系统管理员', 'password': '1111aaa*'},
    'security_admin': {'user': '安全管理员', 'password': '1111aac*'},
    'audit_admin': {'user': '审计管理员', 'password': '1111aab*'},
}

def login(page, role='system_admin'):
    cred = ACCOUNTS[role]
    page.goto('https://192.168.110.243:8443/login?type=pwd')
    page.wait_for_load_state('networkidle')
    # 选择角色下拉
    page.click('.ant-select-selector')
    page.click(f'text={cred["user"]}')
    # 填密码
    page.fill('#normal_login_password', cred['password'])
    # 识别验证码
    code = solve_captcha(page)
    page.fill('#normal_login_vercode', code)
    page.click('button:has-text("登 录")')
    page.wait_for_load_state('networkidle')
    return 'login' not in page.url
```

## 三、测试重点提示

### 综合安全网关特有关注点

1. **三权分立机制**：系统管理员/安全管理员/审计管理员职责隔离是此类产品核心设计，重点验证是否真正隔离
2. **VPN 策略下发**：安全管理员配置的策略是否能被其他角色篡改
3. **审计日志完整性**：敏感操作是否全部记录，日志是否可被删除/篡改
4. **固件升级接口**：是否有签名校验，是否有文件类型限制
5. **网络诊断工具**：ping/traceroute 等内置工具是否有命令注入风险
6. **证书管理**：导入证书功能是否校验格式，是否有路径遍历

### 验证码处理策略

当前验证码为图形验证码（如 `N26G`），推荐：
- **方案1**：Claude Vision API 识别（已有 `analyze_captcha.py` 基础）
- **方案2**：多次截图取均值（验证码较简单时）
- **方案3**：通过 SSH 直接查看服务端生成的验证码值（root权限）
