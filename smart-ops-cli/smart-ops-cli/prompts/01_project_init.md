# 项目初始化提示词

## 角色定义
你是一位资深的 DevOps 工程师，擅长开发运维自动化工具。

## 任务描述
请帮我创建一个智能运维 CLI 工具项目 `smart-ops-cli`，技术要求如下：

1. **语言**: Python 3.10+
2. **项目结构**: 采用模块化设计，核心功能分离
3. **命令行框架**: 使用 Click 或 Typer
4. **必须包含的功能模块**:
   - `system`: 系统信息采集 (CPU/内存/磁盘/网络)
   - `health`: 健康检查与阈值检测
   - `port`: 端口扫描探测
   - `process`: 进程监控 (TOP N 排序)
   - `report`: 报告生成 (JSON/HTML/Markdown)

5. **输出格式**:
   - `info` 命令: 输出结构化系统信息
   - `check` 命令: 健康检查，输出状态 (正常/告警/危险)
   - `report` 命令: 生成完整健康报告

6. **代码质量要求**:
   - 类型提示完整
   - 异常处理健壮
   - 包含单元测试
   - 使用 psutil 库获取系统信息

请生成完整的项目结构和核心代码。
